This is a demonstration of OAuth 2.0. Upon clicking "go to provide google account," this site will prompt you to provide access to your gmail email address via gooogle account login. Page will handle the user getting redirect back from Google's endpoint and handle exchange of access code for a token. First name and last name and the URL to access the Google Plus account will then be displayed along with the state variable that was used to secure the original redirect.
